# BRN PR Board V1.0.0

## ขอบเขตเวอร์ชันนี้

- เปลี่ยนชื่อผลิตภัณฑ์เป็น **BRN PR Board**
- ใช้ปฏิทินขนาดใหญ่เป็นส่วนหลักของหน้าเว็บ
- คลิกวันที่เพื่อเพิ่มงานและแสดงงานในปฏิทิน
- ปรับฟอนต์ ลำดับชั้นข้อความ และระยะห่างให้อ่านง่ายขึ้น
- เพิ่มการเคลื่อนไหวแบบนุ่มนวล โดยไม่รบกวนการใช้งาน
- รองรับมือถือ แท็บเล็ต โน้ตบุ๊ก และจอขนาดใหญ่
- คง Firebase Authentication และการตั้งค่าหลังบ้านเดิมไว้
- ตั้ง Service Worker cache ใหม่เป็น `brn-pr-board-v1.0.0`

## สิ่งที่ยังไม่รวมใน V1

- การ Sync งานผ่าน Firestore เต็มระบบ
- การส่งแจ้งเตือน LINE อัตโนมัติจากฟอร์มหน้าเว็บ
- AI แนะนำแนวทางตามชื่องานแบบอัตโนมัติเต็มรูปแบบ

ส่วนเหล่านี้จะเชื่อมต่อกับหลังบ้านเดิมในเวอร์ชันถัดไป โดยไม่เริ่มระบบ Firebase/LINE ใหม่
