`Authentication → Users → เลือกบัญชี → Reset password / จัดการบัญชี`

## สิทธิ์รอบนี้

ทั้ง admin และ staff ใช้พื้นที่ทำงานเดิมได้ เช่น เพิ่มงาน แก้งาน เปลี่ยนสถานะ และสร้าง Prompt

ตัวแปรสำหรับพัฒนาขั้นต่อไป:

```js
window.BRN_CURRENT_USER
// { uid, email, username, displayName, role }
```

หน้าเว็บยังเก็บข้อมูลงานในเบราว์เซอร์เดิม ขั้นต่อไปจึงควรเชื่อม Cloud Firestore และเขียน Security Rules เพื่อให้ข้อมูลซิงก์ทุกเครื่องและบังคับสิทธิ์ที่ฐานข้อมูลจริง
