# BRN PR Workspace – Calendar Refresh V10.0.0

- ใช้ไฟล์ฐาน `brn-pr-workspace-main(2).zip`
- ปรับปฏิทินเป็นองค์ประกอบหลักด้านบนของหน้า
- เปลี่ยนฟอนต์และลำดับชั้นตัวอักษรให้อ่านง่ายขึ้น
- เพิ่มภาพมาสคอตแบบลอย การ์ดลอย และ animation แบบเบา
- คลิกวันที่ครั้งเดียวเพื่อเปิดหน้าต่างเพิ่มงาน
- แบบฟอร์มหลักเหลือ ชื่องาน รายละเอียด วัน เวลา สถานที่ และเจ้าของงาน
- ซ่อนช่องผู้รับผิดชอบ เพราะใช้ทีมประชาสัมพันธ์ร่วมกัน
- ซ่อนส่วนที่ซ้ำซ้อนจากหน้าแรก: Hero เดิม, Daily strip, สรุปตัวเลข, Board และ Prompt
- คงโค้ดเดิมไว้เพื่อไม่กระทบข้อมูลและระบบ Authentication
- เพิ่ม `refresh.css` และ `refresh.js`
- เปลี่ยน Service Worker cache เป็น `brn-pr-v10-calendar-refresh`
