# BRN PR Board

กระดานปฏิทินงานประชาสัมพันธ์สำหรับเทศบาลเมืองบางรักน้อย โดยให้งานในปฏิทินเป็นศูนย์กลางของทีม และมีระบบช่วยจัดแนวทางการถ่ายภาพ ถ่ายคลิป สิ่งที่ต้องเตรียม และไอเดียคอนเทนต์จากข้อมูลงานจริงแบบอยู่เบื้องหลัง

## สิ่งที่ระบบทำ
1. เพิ่ม แก้ไข และลบงานผ่านปฏิทิน
2. แสดงปี พ.ศ. เดือน วันภาษาไทย วันพระ วันหยุด และวันสำคัญ
3. แยกสีระหว่างวันสำคัญ วันพระ และงานทีม
4. บันทึกงานลง Firestore collection `prEvents`
5. ส่งคิวแจ้งเตือนเข้า `lineOutbox` เมื่อเลือกแจ้ง LINE
6. จัดแนวทางคอนเทนต์จากชื่องาน รายละเอียด ประเภทงาน วันสำคัญ และวันพระ

## ค่าแจ้งเตือนเริ่มต้น
- ทุกงานที่เลือกแจ้ง LINE จะตั้งค่าเตือนอัตโนมัติเป็น `แจ้งล่วงหน้า 1 วัน` และ `แจ้งเช้าวันงาน`
- Frontend บันทึกค่า `reminderEnabled`, `reminderPolicy`, `reminderTimezone`, และ `reminders` ลงใน `prEvents`
- Cloud Functions ฝั่งหลังบ้านสามารถอ่านค่าเหล่านี้แล้วสร้างคิวส่งเข้า `lineOutbox` ได้

## หลังบ้านที่ต้องคงเดิม
- Firebase project `brn-pr-board`
- Firebase Authentication
- Firestore collections เดิม
- Cloud Functions `lineWebhook` และ `sendLineOutbox`
- LINE Messaging / `lineOutbox`
- เอกสารกลุ่ม LINE `system/lineGroup`

## หลักการออกแบบ
- งานและปฏิทินต้องนำเสมอ
- ตัวช่วยคิดคอนเทนต์ทำงานหลังฉาก ไม่แย่งพื้นที่หน้าเว็บ
- หน้าเว็บต้องอ่านง่าย ใช้ได้จริงบนมือถือ แท็บเล็ต โน้ตบุ๊ก และจอใหญ่
