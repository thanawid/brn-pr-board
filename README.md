# BRN PR Board

กระดานปฏิทินงานประชาสัมพันธ์ เทศบาลเมืองบางรักน้อย สำหรับให้ทีมเห็นงานทั้งเดือน เพิ่มงานในวันจริง และใช้แนวทางคอนเทนต์ใต้ปฏิทินช่วยวางแผนถ่ายภาพ/วิดีโอ

## วิธีใช้หลัก

- เปิดหน้าเว็บแล้วดูปฏิทินใหญ่เป็นบอร์ดงานหลัก
- คลิกวันที่ในปฏิทินเพื่อเปิดฟอร์มเพิ่มงานทันที
- กรอกข้อมูลงาน: ชื่องาน รายละเอียด วัน เวลา สถานที่ และเจ้าของงาน
- งานที่บันทึกจะแสดงในช่องปฏิทินของวันนั้น และซิงก์ผ่าน Firestore เมื่อหลังบ้านพร้อม
- คลิกชื่องานในปฏิทินเพื่อดูรายละเอียดหรือแก้ไขงาน
- ใต้ปฏิทินมีการ์ดแนวทางถ่ายภาพ ถ่ายคลิป สิ่งที่ควรเตรียม และไอเดียคอนเทนต์ตามวันที่เลือก
- วันพระ วันหยุด และวันสำคัญยังแสดงในช่องปฏิทิน พร้อมแยกสีจากงานทีม PR

## หลังบ้านที่ใช้

เว็บยังใช้โครงเดิมทั้งหมด:

- Firebase project: `brn-pr-board`
- Authentication
- Firestore
- Collection งาน: `prEvents`
- Collection แจ้งเตือน LINE: `lineOutbox`
- LINE group config: `system/lineGroup`
- Cloud Functions เดิม: `lineWebhook` และ `sendLineOutbox`

เมื่อบันทึกงาน เว็บจะสร้างคิวแจ้งเตือนใน `lineOutbox` โดยอ้าง `system/lineGroup` เพื่อให้ Cloud Functions เดิมส่งเข้ากลุ่ม LINE ต่อไป
