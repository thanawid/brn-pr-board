# BRN PR Content Calendar

เว็บแอปปฏิทินไทยสำหรับทีมประชาสัมพันธ์ เทศบาลเมืองบางรักน้อย

## ใช้ทำอะไร

- เปิดดูปฏิทินไทยปี 2569 พร้อมวันสำคัญและวันพระ
- คลิกวันที่เพื่อดู Day Board ของวันนั้น
- ดับเบิ้ลคลิกวันที่เพื่อเปิดฟอร์มเพิ่มงานทันที
- เพิ่มงานเทศบาล วันของเทศบาล โน้ตทีม หรือไอเดียคอนเทนต์เองได้ พร้อมเวลา สถานที่ เจ้าของงาน และผู้รับผิดชอบ
- ดูคำแนะนำว่าควรถ่ายภาพ/วิดีโออะไรกลับมา
- ดูสรุปว่าวันที่เลือกมีงานอะไร อยู่ที่ไหน ใครเป็นต้นเรื่อง ใครต้องทำ และควรลงช่องทางไหน
- ดูแนวคอนเทนต์รายสัปดาห์และรายเดือน
- ดูงานที่จดไว้ตามสถานะ เช่น ไอเดีย ต้องถ่าย กำลังตัด รออนุมัติ และเผยแพร่แล้ว
- สร้าง Prompt สำหรับนำไปใช้ต่อในทันใจ AI Studio หรือกดคำสั่งเร็ว เช่น คิดโพสต์ สคริปต์คลิป Shot list และเช็กลิสต์หน้างาน

## การเก็บข้อมูล

ข้อมูลที่ทีมเพิ่มเองจะถูกเก็บในเบราว์เซอร์ของเครื่องนั้นด้วย `localStorage`
ยังไม่มีระบบล็อกอินหรือฐานข้อมูลกลาง ถ้าจะให้หลายคนเห็นข้อมูลเดียวกัน ควรต่อฐานข้อมูลหรือ Google Sheet ในเวอร์ชันถัดไป

## เปิดใช้งาน

เปิด `index.html` ได้โดยตรง หรือรันเซิร์ฟเวอร์เล็กๆ:

```bash
npm start
```

แล้วเปิด `http://localhost:4173`
