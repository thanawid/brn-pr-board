# BRN PR Board

กระดานปฏิทินงานประชาสัมพันธ์ เทศบาลเมืองบางรักน้อย พร้อมระบบจดงานและคิวแจ้งเตือน LINE

## วิธีใช้หลัก

- คลิกวันที่ในปฏิทินเพื่อดูงานของวันนั้นทางแผงขวา
- ดับเบิ้ลคลิกวันที่ หรือกด `เพิ่มงาน` เพื่อเปิดฟอร์มละเอียด
- ใช้กล่อง `จดงานเทศบาล` ทางขวาเพื่อเพิ่มงานเร็ว พร้อมเวลา สถานที่ เจ้าของงาน ผู้รับผิดชอบ และประเภทงาน
- เลือก `แจ้งเตือนเข้า LINE` เพื่อให้เว็บสร้างคิวแจ้งเตือนให้หลังบ้าน
- กด `ส่ง LINE` จากรายการงาน หรือ `ส่ง LINE ตอนนี้` ในหน้ารายละเอียด เพื่อสร้างคิวแจ้งเตือนทันที
- ด้านล่างปฏิทินมีแนวทางถ่ายภาพ ถ่ายคลิป สิ่งที่ต้องเตรียม และไอเดียคอนเทนต์ตามงานของวันที่เลือก

## อัปโหลด GitHub Pages

นำไฟล์ทั้งหมดในโฟลเดอร์นี้วางที่ root ของ repository `brn-pr-board`

## โครงข้อมูลที่เว็บใช้

- `prEvents` — งานในปฏิทิน
- `lineOutbox` — คิวแจ้งเตือน LINE สำหรับหลังบ้าน

ตัวอย่าง field ใน `lineOutbox`:

- `kind`: `line-reminder` หรือ `line-immediate`
- `status`: `pending`
- `target`: กลุ่มหรือผู้รับ LINE
- `sendAt`: เวลาแจ้งเตือนสำหรับ Cloud Functions
- `message`: ข้อความที่พร้อมส่ง
- `event`: ข้อมูลงาน เช่น ชื่อ วันที่ เวลา สถานที่ ผู้รับผิดชอบ

ตั้งค่า LINE ได้ใน `auth-config.js` ที่ `lineReminder` ถ้ามี Cloud Functions ฟัง Firestore อยู่ให้ใช้ค่าเดิมแบบ `firestoreOutbox` ได้เลย หากต้องการยิง HTTP endpoint ให้ใส่ `webhookUrl`

หาก Firestore rules ยังไม่อนุญาต `prEvents` เว็บจะทำงานด้วย localStorage บนเครื่องนั้นชั่วคราว และเก็บคิว LINE สำรองไว้ในเครื่อง
