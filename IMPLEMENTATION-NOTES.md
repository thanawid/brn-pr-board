# Implementation Notes

## แนวคิดการออกแบบ

หน้าตาปัจจุบันตั้งใจให้สงบ อ่านง่าย และใช้สีเป็นระบบสถานะ ไม่ใช้สีเพื่อการตกแต่งอย่างเดียว เมื่อ workflow คงที่แล้วสามารถเพิ่มความสดและเอกลักษณ์แบรนด์ได้โดยปรับ design tokens ใน `:root` ของ `styles.css` โดยไม่ต้องแก้โครงสร้างหน้า

## ลำดับการเพิ่มความน่าใช้รอบถัดไป

1. ปรับชุดสีและไอคอนให้มีบุคลิก BRN ชัดขึ้น
2. เพิ่ม micro-animation เฉพาะการเปลี่ยนสถานะและการเปิดใบงาน
3. เพิ่ม cover image/thumbnail เฉพาะงานที่มีสื่อแล้ว
4. เพิ่ม dark mode หากทีมใช้งานกลางคืนจริง
5. เชื่อมปุ่มส่งบรีฟไป Tanjai AI Studio เมื่อกำหนดสัญญาข้อมูลระหว่างสองระบบแล้ว

ไม่ควรเพิ่มกราฟหนัก Sidebar ซับซ้อน หรือ AI Chat เต็มหน้า เพราะจะทำให้แกน “งานนำ” อ่อนลง
